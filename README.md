# Library Articles

Companion static site for https://huggingface.co/spaces/helwyr55/library — renders full JSX React article artifacts with Recharts, custom SVG, and magazine-quality typography.

Articles are produced in Claude Desktop using the `technical-ds-article`, `natgeo-article`, and `travel-photography-article` skills, then published here via the `library-article-publisher` skill.

Run `python3 build.py` after adding a JSX file to regenerate HTML wrappers.
